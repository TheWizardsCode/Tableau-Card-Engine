## Playing Card Assets

- **ace_of_spades.svg** - Original SVG created for this project. CC0 (Public Domain).
- **card_back.svg** - Original SVG created for this project. CC0 (Public Domain).

These are placeholder assets created as simple SVGs for the hello-world
toolchain verification scene. They are intended to be replaced with a
complete playing card set from a CC0/public domain source such as:

- [OpenGameArt.org](https://opengameart.org/) - Various CC0 card sets
- [Kenney.nl](https://kenney.nl/) - CC0 game assets including card sets
- [SVG-cards](https://github.com/htdebeer/SVG-cards) - LGPL-2.1 SVG card deck

When sourcing replacement assets, ensure the license is permissive
(CC0, MIT, Apache 2.0, or similar) and suitable for commercial use.
